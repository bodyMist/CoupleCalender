# CoupleCalender
2021 summer vacation personal project

## 
 개발환경: C# .NET Framework && UWP Library
 
 개발도구: Visual Studio 2019
 
 DATABASE: MYSQL
 
 소프트웨어 형식 : Windows APP

## 기능
### 일반기능
1. 로그인
2. 회원가입
3. 커플 신청/응답
### 커플기능
1. 일정추가(공유)
2. 이벤트 DAY 설정(D-Day || 사귄일수)
3. 달력에 일정 표기해서 보여주기
4. 스토리 CRUD
